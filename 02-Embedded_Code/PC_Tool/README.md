# MyFOC_NFlux PC Tool

这是 MyFOC_NFlux 的图形化调试上位机，用来替代 VOFA 的基础观察和控制流程。

设计上参考了常见开源遥测仪表盘/串口示波器的组织方式：左侧放连接、控制、通道和记录，右侧放实时数据卡片和多曲线窗口。当前实现保留 MyFOC_NFlux 自己的简单协议，不引入复杂工程文件。

界面技术栈：

- `PySide6`：Qt 图形界面。
- `pyqtgraph`：实时曲线、鼠标缩放、拖拽和平移。
- `pyserial`：串口通信。

## 功能

- 串口选择、刷新、连接、断开。
- 串口列表每 1 秒自动刷新，插拔 CH340 后不用重启上位机。
- 实时解析 STM32 上传的 VOFA JustFloat 兼容数据。
- 8 路遥测卡片显示：
  - `ia`
  - `ib`
  - `ic`
  - `FluxTheta`
  - `FluxWm`
  - `RefSpeed`
  - `rtU.v_bus`
  - `vbus_raw`
- 曲线区为 2x2 矩形布局：三相电流、磁链角、速度、母线电压。
- 可勾选要显示的曲线。
- 三相电流放在同一个电流窗口里，点击图例或左侧通道复选框可以隐藏/恢复某一相。
- 磁链角卡片显示原始 `0..2π` 角度，曲线显示展开后的连续角度趋势。
- 支持浅色/深色主题切换。
- 鼠标滚轮缩放曲线，拖拽平移曲线。
- 支持跟随最新数据、Y 轴自动缩放、重置视图、暂停绘图。
- 支持清空曲线缓存，重新从当前时刻观察波形。
- 下发启动/停止命令。
- 下发速度给定。
- 自定义命令发送窗口。
- CSV 数据记录。
- 演示数据模式：不接板子也能预览界面和曲线。
- 预留了位置控制、轨迹规划和参数辨识页面，后续协议扩展时再启用。

注意：当前上位机控制版固件上电后只做电流 offset 标定，不会自动启动电机；点击“启动”后才会下发 `RUN=1`。

## 运行

双击：

```text
run.bat
```

`run.bat` 会自动创建 `.venv` 并安装依赖。

也可以手动运行：

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
.\.venv\Scripts\python.exe myfoc_host.py
```

## 曲线操作

- 鼠标滚轮：缩放。
- 鼠标左键拖拽：平移。
- 接收数据时滚轮缩放/拖拽会自动取消“跟随最新数据”，避免界面马上把缩放范围拉回去。
- 勾选“跟随最新数据”：曲线窗口自动跟随最新采样点。
- 取消“跟随最新数据”：冻结当前时间窗口，可以自由查看历史波形，不会清空曲线。
- 勾选“Y 轴自动缩放”：根据当前数据自动调整纵轴。
- 点击“重置视图”：恢复跟随和自动缩放。
- 点击“清空曲线”：丢弃旧缓存，重新从当前接收数据开始画。
- 左侧通道勾选框：选择要显示的数据。

## 默认串口参数

- 波特率：`2000000`
- 数据位：`8`
- 校验：`None`
- 停止位：`1`

## 协议

见 [protocol.md](protocol.md)。

STM32 上传帧：

```text
float[8] + 00 00 80 7F
```

PC 下发命令：

```text
RUN=1
RUN=0
SPD=600
```

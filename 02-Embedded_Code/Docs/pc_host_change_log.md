# 上位机变动记录

本文记录 `02-Embedded_Code/PC_Tool` 上位机的界面、绘图和交互变更。纯技术根因问题仍记录在 `technical_issue_log.md`。

## 2026-06-17 绘图区与交互体验重构

### 主要变化

- 将右侧波形区重构为上下两幅主图：
  - 上图：`State / Control`，用于显示 `θ`、`Speed`、`Ref`、`T1/T2/T3`、`FOC`。
  - 下图：`Current / Voltage`，用于显示 `Ia/Ib/Ic`、`Id/Iq`、`Id_ref/Iq_ref`、`Ud/Uq`、`Vbus`。
- 每幅图左侧改为纵向彩色胶囊按钮选择通道，图内不再显示 legend，曲线识别依赖胶囊颜色和简称。
- 绘图恢复为单 Y 轴显示，由当前可见曲线自动缩放；避免多 Y 轴导致数值堆叠和轴线混乱。
- 顶部实时遥测摘要卡片移除，减少快速变化数据对视觉区域的占用。
- 图表控制按钮移动到主界面右上角，包括暂停、CSV 记录、重置视图、清空曲线、主题切换和帧率统计。
- 暂停、记录、重置、清空和主题切换按钮改为 Qt 绘制的矢量图标，不再依赖文字或 emoji。
- 通道胶囊由隐藏指示器的 `QCheckBox` 改为整块可点击的 `QPushButton`，点击矩形右侧空白区域也能切换曲线。
- 浅色/深色主题下的图表网格透明度统一配置，浅色主题保留可见的低对比度网格。
- 左侧控制区压缩高度，串口和波特率下拉框改为扁平风格，速度输入和窗口输入去掉老式上下箭头。
- 去掉“诊断遥测”页签，保留解析、CSV 和绘图通道能力。

### 行为修正

- 手动滚轮缩放某个图后，该图不再自动缩回最新窗口。
- 手动缩放只影响当前图，不影响另一幅图。
- 隐藏通道重新启用后，会立即使用已有历史数据刷新曲线。
- `Speed` 和 `Vbus` 绑定到当前单轴绘图路径，避免选择后无曲线显示。
- `Iq_ref` 数据链路已核对为固件 `load_data[10] = FocDiagIqRef` 到 PC 解析 `iq_ref`，并增加测试防止顺序错位。
